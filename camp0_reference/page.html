The Value of Coding

Coding is not the solution to a problem. Coding is a tool which helps us solve problems and create things. Sometimes, pretty awesome things. Like Angry Birds.

Coding at its core is the way we speak to computers. How we give a computer instructions. Think about the world in 20 years. We’ll be sitting in driverless cars and moving objects with our mind. The one skill you had better have is to be able to read and write code.

Unless you intend to be a mountain hermit, technology will be present in your life one way or another. Don’t just be a consumer of technology. Understand it. Learn how to control it.
